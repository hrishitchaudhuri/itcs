# mini-proj-traffic

## Note

For RL, switch to branch RL_1

## Use Makefile to create trips and routes

1. Convert osm to net.xml

    ```shell
    make osm
    ```

2. Create trips

    ```shell
    make car_trips
    ```

3. Run simulation in sumo-gui

    ```shell
    make sim
    ```

4. Clean

    ```shell
    make clean
    ```
