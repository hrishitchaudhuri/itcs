# mini-proj-traffic

## Use Makefile to create trips and routes

1. Convert osm to net.xml

    ```shell
    make osm
    ```

2. Create trips

    ```shell
    make trips
    ```

3. Run simulation in sumo-gui

    ```shell
    make sim
    ```

4. Run python Round Robin script

   ```shell
   make rr
   ```

5. Clean

    ```shell
    make clean
    ```
